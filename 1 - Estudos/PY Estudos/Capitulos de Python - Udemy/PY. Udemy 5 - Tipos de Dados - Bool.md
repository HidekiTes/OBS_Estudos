```python
# Tipos de Dados: Bool

"""
Tipo de dado Bool (boolean)
Ao questionar algo em um programa, só existem duas respostas possíveis
Sim (True) ou Não (False)
Existem vários operadores para "questionar"
Dentre elas o ==, que é um operador lógico que questiona
se o valor é igual ao outro
"""

print(10 == 10) # Sim => True
print(10 == 11) # Não => False
print(type(True))
print(type(False))
print(type(10 == 10))
print(type(10 == 11))
```
