## Explicação
Além do Hello World, podemos executar outros códigos que podem retornar informações diferentes
## Código 
```js 
//Código 1 - Para imprimir o Olá, mundo! string, digite o seguinte no console:
console.log("Hello, World!");

//Código 2 - Você também pode realizar operações matemáticas no Console: 
console.log(2 + 6);

//Código 3 - Você também pode tentar uma matemática mais complicada:
console.log(34348.2342343403285953845 * 4310.23409128534);

//Código 4 - Além disso, você pode trabalhar em múltiplas linhas com variáveis:
let today = new Date();
console.log("Today's date is " + today);
```
